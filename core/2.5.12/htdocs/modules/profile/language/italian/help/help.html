Aiuto: Profilo

DESCRIZIONE

Il modulo Profilo serve per gestire i campi personalizzati del profilo utente.

INSTALLAZIONE/DISINSTALLAZIONE

Nessuna misura speciale necessaria, seguire la procedura di installazione standard – estrarre la cartella /profile nella directory ../modules. Installare il modulo tramite Amministrazione -> Modulo di Sistema -> Moduli.

Istruzioni dettagliate sull'installazione dei moduli sono disponibili nel Manuale Operativo di XOOPS

ISTRUZIONI OPERATIVE

Per configurare questo modulo è necessario:

  • Configurare le preferenze per il modulo (vedi ‘Preferenze’) e opzionalmente il blocco Profilo se si intende utilizzarlo (vedi ‘Blocchi’).
  • Modificare le Categorie esistenti o aggiungerne di nuove.
  • Modificare i Campi esistenti o aggiungerne di nuovi. Qui è possibile specificare quali campi saranno visibili in quale categoria e se saranno visibili durante la registrazione dell'utente.
  • Definire l'ordine dei passaggi di Registrazione.
  • E infine, è possibile impostare i permessi per i singoli campi - quali sono modificabili, quali sono ricercabili.

Misure Anti-Spam

Per minimizzare le registrazioni di spam, fare quanto segue:

  • andare al modulo Protector in Amministrazione, andare a Preferenze e poi in basso, a questa opzione: "Stop Forum Spam" Controlla i dati POST rispetto agli spammer registrati nel database di www.stopforumspam.com.
    Impostarlo su "Banna l'IP (nessun limite)"

  • in /class/captcha/config.php, assicurarsi che la modalità sia impostata su "text":

    return $config = array(
    'disabled' => false, // Disabilita CAPTCHA
    'mode' => 'text', // modalità predefinita, è possibile scegliere 'text', 'image', 'recaptcha'(richiede chiave api)
    'name' => 'xoopscaptcha', // nome captcha
    'skipmember' => true, // Salta il controllo CAPTCHA per i membri
    'maxattempts' => 10, // Tentativi massimi per ogni sessione
    );

  • Nel modulo Profilo, andare ad Amministrazione e nel passaggio Base situato in:
    /modules/profile/admin/step.php?id=1
    impostare "Salva dopo lo stop" su "No"


  • Nelle Preferenze del Profilo, impostare l'opzione "Usa Captcha dopo il secondo passaggio di Registrazione" su "Sì" (è l'impostazione predefinita)
