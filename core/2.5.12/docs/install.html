Installazione di XOOPS

Prima installazione

Prefazione:

XOOPS è un sistema di pubblicazione web open-source orientato agli oggetti scritto in PHP. È uno strumento ideale per lo sviluppo di siti web dinamici per comunità di piccole e grandi dimensioni, portali aziendali, portali corporate, weblog e molto altro.

XOOPS è rilasciato sotto i termini della GNU General Public License (GPL) versione 2 o successiva, ed è gratuito da usare e modificare. È gratuito da ridistribuire a condizione che tu rispetti i termini di distribuzione della GPL.

Requisiti

  • Server WWW (Apache, NGINX, IIS, ecc.)
  • PHP 5.6.0 o superiore, 7.4+ raccomandato
  • MySQL 5.6 o superiore, MySQL 8.0+ raccomandato

Prima dell'installazione

  1. Configura correttamente il server WWW, PHP e il server di database.
  2. Prepara un database per il tuo sito XOOPS.
  3. Prepara un account utente e concedi all'utente l'accesso al database.
  4. Rendi scrivibili le directory e i file: %s
  5. Per considerazioni di sicurezza, ti consigliamo vivamente di spostare le due directory sottostanti fuori dalla document root e di cambiare i nomi delle cartelle: %s
  6. Crea (se non già presenti) e rendi scrivibili le directory: %s
  7. Attiva i cookie e JavaScript del tuo browser.

Installazione in locale

Se stai eseguendo un ambiente locale per lo sviluppo o il testing, assicurati di aver soddisfatto i requisiti precedenti. Una volta fatto ciò, copia il contenuto della directory htdocs (dal file di distribuzione di XOOPS) nel percorso della document root del tuo ambiente web. Una volta copiati i file, puoi avviare l'installazione puntando il browser su "http://iltuosito.com". Questo avvierà il processo di installazione.

Installazione su una piattaforma ospitata

Se stai eseguendo l'installazione in un ambiente ospitato, decomprimi i file di XOOPS localmente o sul server se hai accesso shell (es. SSH). Una volta fatto, assicurati di spostare o copiare tutti i file di XOOPS dalla directory htdocs alla tua root web directory (il tuo provider di solito fornisce questa posizione con le istruzioni). Una volta copiati i file, puoi avviare l'installazione puntando il browser su "http://iltuosito.com". Questo avvierà il processo di installazione.
